ますこっととばし

なにこれ？

マスコットキャラクターがとびます。

動作環境

Windows + .NET Framework 4 で動きます。

Windows 7 と 8.1、10 で動作確認しています。Vista はたぶん動きます。XP は知りません。

Mono？ いえ、知らない子ですね。

インストール／アンインストール

zip ファイルを解凍して好きな場所に置いてください。

いらなくなったらフォルダごと削除してください。

使い方

MascotTobashi.exe を実行してください。

タスクトレイのアイコンを右クリックするといろいろ設定したり終了したりできます。

マスコットは一定時間毎に勝手にとんできます。
タスクトレイのアイコンをクリックしてもマスコットがとんできます。

とんできたマスコットをクリックすると新たなマスコットがとんできます。

FAQ

マスコットがとぶと処理が重くなる。動きがカクカクする。

マスコットをとばす頻度を下げるか、表示倍率を小さくすると改善するかもしれません。
CPU にやさしくないです。ごめんなさい。

範囲設定が画面外に行ってしまった。

アプリケーションを終了してから Config.xml を削除してください。設定がリセットされます。
範囲設定を開いて Windows + Shift + ←→ でもなんとかなるかもしれません。

とんでくるマスコットの種類を増やしたい／減らしたい。

アプリケーションを終了してから images フォルダ直下の画像を増やして／減らしてください。
拡張子は png、gif、jpg、jpeg、bmp に対応しています。
アニメーション GIF には対応していません。

使用キャラクター

いずれのキャラクターの画像も再使用は禁止です。
